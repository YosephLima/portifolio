# Meu Portifólio

Portifólio desenvolvido no curso de Desenvolvimento de Software Multiplataforma (FATEC SJC).

**Tecnologias**
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express, Zod, Nodemon

**Configuração**

```bash
# baixe o projeto no seu computados
git clone https://github.com/YosephLima/portifolio.git

# acesse a página do projeto
cd portifolio

# instale as dependências do projeto
npm install
```

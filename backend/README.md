## Backend

API REST simples para meu portfólio, com validação por schema e dados em memória.

Tecnologias:
- Node.js
- Express
- Zod
- Nodemon

Documentação:
- [Modelo de dados](docs/modelo_de_dados.md)
- [Detalhes das rotas](docs/detalhes_rotas.md)

## ▶️ Como rodar (backend)

**Requisitos**
- Node.js (LTS)
- npm

**Comandos**
> Antes de rodar, confirme se o terminal está na pasta `/backend`.

```bash
npm run dev
```

Se tudo der certo, o backend estará rodando em `http://localhost:3000`.
